====================================================
Whiteboard with php and websockets
====================================================

How to test it on your computer : 

 - Download all this repo and put it in a new file in www/ directory of easyphp
 - Make sure all your paths points to php/ and apache/bin
 - With your console, navigate to sockets/ directory and : php -q server.php
 - Open 2 windows of Chrome, navigate to http://localhost
 - Have fun with whiteboards
